# YAOF — Yet Another Obtainium Fork

> ⚠️ **Notice: This project is archived and no longer maintained. (本專案已停止維護與更新)**

**YAOF** 是基於 Android Jetpack Compose 與 Material Design 3 開發的開放原始碼應用程式更新追蹤工具（概念源自 Obtainium）。

---

## ✨ 已實現功能 (Features)

- 🎨 **Material You 現代化介面**：支援系統動態取色 (Monet Dynamic Colors) 與深色模式。
- 🌐 **多語言支援**：支援跟隨系統、繁體中文、簡體中文、English、日本語。
- 🔍 **多來源介面預留**：支援 GitHub, GitLab, F-Droid, Codeberg, IzzyOnDroid 及 Direct URL 解析結構。
- ⚙️ **架構過濾**：支援根據 CPU 架構 (如 `arm64-v8a`) 自動篩選 APK。
